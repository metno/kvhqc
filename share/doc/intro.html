Keyboard shortcuts

Some keyboard shortcuts are indicated by underlined letters, for example Søk for the key compination Alt+S. In addition, the following shortcuts are defined:

Shortcut Context Action
Ctrl+F Search Window Show error list
Ctrl+Shift+R Search Window Show recently visited observations
Ctrl+1 Search Window Show auto list/series
Ctrl+2 Search Window Show station data
Ctrl+3 Search Window Show single observation
Ctrl+G Single Observation Accept original value
Ctrl+H Single Observation Accept model value
Ctrl+J Single Observation Accept corrected value
Ctrl+K Single Observation Reject observation
Ctrl+Z * Undo
Ctrl+Shift+Z * Redo
Ctrl+S * Save changes
